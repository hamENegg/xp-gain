Firebase Setup (Owner Only)

  1. Go to Firebase Console and create a new project.
  2. Enable Firestore Database.
  3. In Project Settings, add a new Web App and copy the config snippet.
  4. Replace the placeholder in js/feedback.js with your config.
  5. Deploy your site. Only you will have access to the database.
